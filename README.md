# ConvertHub ‚Äî All-in-One File Transformer

ConvertHub is a high-performance, secure, and intuitive web application designed for comprehensive document and image transformations. It features a suite of 17 specialized tools ranging from PDF manipulation to OCR and image conversion.

## üöÄ Key Features

- **PDF Core**: Merge, Compress, Split, Organize, and add Page Numbers.
- **Security & Repair**: Protect (Password), Unlock, Watermark, and Repair corrupted PDFs.
- **Transformations**: Markdown to PDF, Word to PDF, Web to PDF, and Image to PDF.
- **Advanced Export**: PDF to Word (DOCX), PDF to Image (PNG/ZIP), and OCR (Optical Character Recognition) to Text.
- **Visualizations**: Markdown to Graph (SVG).

## üíé Technical Excellence

- **Robust Download System**: Uses a custom server-side API with `Content-Disposition` headers to ensure correct file extensions and filenames across all browsers.
- **Modern Stack**: Built with Next.js, Tailwind CSS, and optimized for performance.
- **Privacy First**: Most processing happens directly in the browser or via ephemeral server-side stashing.
